建站参考

http://sunwhut.com/2015/10/30/buildBlog/

快捷键

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new
还能组合使用，如：
hexo d -g (在git上deploy)
hexo s -g (在本地浏览器观看)

更换收藏夹图标

hexo/source/下放置自己的收藏夹图标，但不要命名为favicon.ico
收藏夹图标来源：http://www.easyicon.net/language.en/
