快捷键

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new 新建文件
还能组合使用，如：
hexo d -g (在git上deploy)
hexo s -g (在本地浏览器观看)

本地预览草稿，但不发布

如下方式启动server：

$ hexo server --drafts

下面这条命令可以把草稿变成文章，或者页面(注：filename是不包含.md的部分)：

$ hexo publish [layout] <filename>
